# 更新日志

## 版本 1.1 (2026-01-18)

### 修复的问题

1. **修复 ANSI 颜色代码乱码**
   - 修复下载进度显示中的乱码问题
   - 移除了 yt-dlp 输出中的特殊字符和颜色代码
   - 现在进度信息显示清晰，没有乱码

2. **修复按钮文字乱码**
   - 将"下载中..."改为"Downloading..."
   - 避免 tkinter 中文显示问题

3. **修复 ffmpeg 路径问题**
   - 添加了本地 ffmpeg 路径配置
   - 自动检测并使用本地安装的 ffmpeg
   - 即使系统 PATH 未更新也能正常工作

4. **修复批处理文件编码问题**
   - 创建了无中文字符的启动脚本
   - 避免 Windows 命令提示符编码问题

### 新增功能

1. **智能 ffmpeg 检测**
   - 自动查找系统 PATH 中的 ffmpeg
   - 自动使用本地安装的 ffmpeg
   - 无需手动配置

2. **改进的日志输出**
   - 所有日志输出都会自动清理控制字符
   - 更清晰的进度显示

### 使用方法

#### 启动程序（推荐）
双击运行: **start_gui.bat**

#### 或使用命令行
```bash
python video_downloader_gui.py
```
